The analysis process first uses a strict analyzer, then a relaxed analyzer is applied to what the strict analyzer fails on.

Caveat lector: the most current version of the relaxed analyzer here is too large to host on this site (as of 8/21/2024). The version posted here is a slimmed down older version and in some cases will misanalyze forms that are correctly analyzed by the current version.

Your data is private here! When you analyze words/texts on this site, none of that is shared with (or even seen by) the host. Here's why:

Your browser makes a self-contained file system in your computer's memory when it visits a page. All this site does is (1) load an analyzer into this file system, and (2) tell your computer how to use the analyzer. When you analyze some text, it goes into the file system to be processed. In short, all the analysis work is done on your computer, and none of what you analyze is sent to a server somewhere for processing/storage. This is why this service is free (yay!) and your data is as secure as your browser (which is probably pretty secure).
